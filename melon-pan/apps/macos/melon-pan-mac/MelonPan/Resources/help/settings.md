# Settings

Settings is a separate macOS window opened with Command-Comma or from the sidebar Settings pane.

Account settings cover Google sign-in state. Sync settings cover runtime behavior and recovery choices. Editor settings cover text editing preferences such as Vim mode.

Storage settings show local cache behavior. Drive settings cover root folder filtering. Diagnostics and recovery controls help inspect state or clear local data when a development build gets wedged.

Changes should affect local app behavior only; Google Docs remains the source of remote document truth.
