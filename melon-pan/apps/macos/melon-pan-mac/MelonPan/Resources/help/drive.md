# Drive

Drive shows cached Google Drive folders and Google Docs files. Refresh the Drive tree after sign-in or when a document does not appear.

Open a Doc by selecting it from the Drive pane, by using the command palette, or by using a Melon Pan deep link when one is available.

Root scoping keeps the Drive view focused on the folder tree Melon Pan is allowed to show. Non-Doc files are hidden by default because the macOS editor only opens editable Google Docs.

If Drive looks empty, verify Google sign-in, refresh the tree, then inspect Diagnostics for cache and account status.
