# Markdown Syntax

Use headings with one or more number signs, for example # Title, ## Section, and ### Subsection.

Use **bold**, *italic*, `inline code`, links, ordered lists, unordered lists, block quotes, fenced code blocks, and horizontal rules as normal Markdown.

Melon Pan translates Markdown into Google Docs structural updates during push. Keep documents simple when you need predictable Docs output: headings, paragraphs, lists, links, emphasis, and code spans are the safest surface.

When importing or pasting Markdown, review the rendered Google Doc after the first push if the source uses tables, custom HTML, or unusual nested list structure.
