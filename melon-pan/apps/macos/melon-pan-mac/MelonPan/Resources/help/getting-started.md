# Getting Started

Melon Pan is a Markdown-first Google Docs editor. Open or create a draft, edit Markdown locally, then push changes back to Google Docs when you are ready.

Use the sidebar to move between Home, Drive, Conflicts, Diagnostics, and Settings. Use the command palette for fast actions when you do not want to leave the keyboard.

Start with a local draft when you are offline or sketching. Open a Google Doc from Drive when you want the editor state to round-trip through the Docs API.

Diagnostics shows cache paths, account state, and recent sync health. It is the first place to check when sign-in, Drive refresh, or push behavior looks wrong.
