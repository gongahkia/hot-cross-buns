# Keyboard Shortcuts

Use Command-P to open the command palette. It searches commands, panes, cached Drive documents, and sync actions such as Push, Pull, and Drain.

Use Command-N to create a new local draft. Use Command-W to close the active editor tab. Use Command-1 through Command-5 to switch between Home, Drive, Conflicts, Diagnostics, and Settings.

Inside the editor, Command-S pushes queued rich-document operations through the Docs batchUpdate path. Standard editing shortcuts such as Command-B, Command-I, Command-K, Command-F, Command-G, Command-Shift-G, Command-Z, and Command-Shift-Z follow the native editor surface.

Tab indents and Shift-Tab outdents where the editor accepts indentation.
