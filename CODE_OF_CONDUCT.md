# Code Of Conduct

This project expects direct, technical, respectful collaboration.

## Expected Behavior

- Keep discussion focused on the work.
- Critique code, designs, evidence, and tradeoffs rather than people.
- Assume public comments may be read by future users and contributors.
- Redact private data, credentials, logs, and screenshots before posting.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Publishing private information or credentials
- Repeated off-topic or bad-faith comments
- Abuse of issues, pull requests, discussions, or security reporting channels

## Enforcement

Maintainers may edit, hide, lock, or delete comments; close issues or pull requests; block users; or report abuse to GitHub when needed.

Security reports belong in [SECURITY.md](SECURITY.md), not public issues.
