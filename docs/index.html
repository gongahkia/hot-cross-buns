First launch on macOS

One quick thing before you download.

Preview DMGs are unsigned for now, so macOS may ask you to approve Hot Cross Buns once before it opens normally. Google sync asks for your own Desktop OAuth client during setup.

  • Move Hot Cross Buns into your Applications folder after opening the DMG.
  • Try opening the app once.
  • If macOS blocks it, go to System Settings > Privacy & Security and click Open Anyway.
  • You should only need to do this once per Mac.

You'll download the latest DMG directly.

Checksums are published for each release. See Privacy and Trust and the GitHub README for OAuth setup.
