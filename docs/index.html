Designed with security in mind

Your plans,
under lock and key.

A planner knows where you'll be, who you'll meet, and what you haven't done yet. That is sensitive data, and Hot Cross Buns treats it that way. Every surface is hardened on purpose — not after the fact.

AES-256-GCM on disk. Secrets in Keychain, never plaintext. URL handlers gated. Regex bounded. Logs redacted.

  • Local cache encryption

    Opt-in AES-256-GCM wrapping over the on-disk cache, keyed by a passphrase-derived CryptoKit.SymmetricKey. Nothing readable by another process on the machine.

  • No eval, no ReDoS

    Natural-language parsers dispatch through typed enums, not string eval. All regex uses \b anchors and bounded quantifiers — no catastrophic-backtracking surface.

  • Trusted IPC only

    Share-extension and Services-menu payloads carry a source bundle-id, validated against the app prefix, with a 10‑minute trust window and an 8 KB byte cap.

  • Logs don't leak

    Structured os_log interpolations — bare message public, metadata private. Nothing useful for a nosy Console.app observer.

The details

Small things, seriously considered.

Find anything, in a sentence

Ask the way you think. attendee:alex duration>30m has:notes tag:work due<+7d is one phrase, not six filters and a wall of checkboxes. Save the ones you run weekly.

Write the recurring task once

Your weekly review, your standup prep, your project kickoff — each is a template that fills itself in. {{today}}, {{+7d}}, {{clipboard}}, {{prompt:Label}}. You stop retyping the same thing Monday mornings.

Fits inside your existing flow

Call it from Raycast, Spotlight, Alfred, Shortcuts, or any script — the hotcrossbuns:// URL scheme and macOS Services hook straight into the launchers you already use. The task is captured before you've even switched windows.

Reads the way you want it to

A serif for long‑form notes. A mono for the query bar. A tighter sans for dense lists. Six surfaces, six independent font choices — because one font never fits every job.

Keeps working in the tunnel

Planes, trains, bad hotel wifi — type as if you were online. A local mutation queue holds every change and reconciles it cleanly (etag‑based, no silent overwrites) the moment you're back on a signal.

Matches the rest of your desktop

If your terminal, editor, and window manager already wear Catppuccin, your planner finally does too. Latte and Mocha, tuned by hand — not a generic Light/Dark auto‑invert.
